# Simple-Web-Calculator
A simple Calculator for general purposes.

## Why use it?
Its useful for simple calculations.

## Features

* Add
* Multiply
* Subtract
* Divide

## Built With

* LOVE
* HTML
* CSS
* JavaScript
* C#
* ASP.NET CORE

## How to Run?

To run my application you simply need to clone the project and run the Basic Web Project.sln file.
